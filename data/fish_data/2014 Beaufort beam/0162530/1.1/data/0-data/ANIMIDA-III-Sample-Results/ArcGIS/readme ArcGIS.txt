Introduction
============

The folder "ANIMIDA_III_Sampling_Results.gdb" is an Esri file geodatabase which contains sampling results for the 2014 and 2015 ANIMIDA III cruises.  You can view the geodatabase using ArcGIS version 10.1 or greater, or other Geographic Information System (GIS) software capable of viewing an Esri file geodatabase version 10.1.

The DataValues table contains the results, while the other tables and feature classes provide additional metadata about the results. See "ERD and Data Dictionary.pdf" for a description of each dataset.
